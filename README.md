# Каталог товаров

Тестовое задание: одностраничное приложение (SPA) на React для отображения списка товаров с возможностью поиска и просмотра деталей в модальном окне.

## Основной функционал
- Вывод списка товаров в виде адаптивной сетки.
- Динамический поиск по названию ("на лету").
- Просмотр подробной информации о товаре в модальном окне (используется `ReactDOM.createPortal`).
- Модальное окно поддерживает закрытие по кнопке, клику по оверлею и нажатию `Esc`. При открытии блокируется скролл основной страницы.
- Адаптивная верстка (мобильные устройства, планшеты, десктоп).
- Данные подгружаются асинхронно из мок-сервиса (имитация API-запроса).

## Стек технологий
- **Фреймворк:** React (Hooks) + Vite
- **Стили:** CSS Modules
- **Архитектура:** Упрощенный FSD (Feature-Sliced Design)
- **Инструменты:** pnpm, Prettier, Lefthook (pre-commit форматирование)

## Структура проекта
Код организован по методологии FSD:
- `src/app` — точка входа и глобальные стили.
- `src/pages` — страница каталога.
- `src/widgets` — виджет списка товаров.
- `src/entities` — карточка товара, детали товара, UI-компонент модалки и mock-сервис.

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/KonnovEgorIDB/task-ecom.git
   ```
2. Установите зависимости (используется pnpm):
   ```bash
   pnpm install
   ```
3. Запустите dev-сервер:
   ```bash
   pnpm dev
   ```
